# Product Overview

LUKA is an e-commerce clothing store web application offering fashion products for men and women.

## Key Features

- User authentication (login/registration)
- Gender-specific shopping sections (Men/Women)
- Product categories:
  - Men: Jeans, Shirts, Sweaters, Accessories
  - Women: Denim, Dresses, Tops, Accessories
- Product browsing with filtering options (size, color, price)
- Product detail modals with size selection
- Shopping cart functionality
- Newsletter subscription

## User Flow

1. Landing page with gender selection (Men/Women)
2. Category-specific main pages with featured products
3. Individual product category pages with grid layouts
4. Product detail modals for adding items to cart
5. User account management through login/registration
